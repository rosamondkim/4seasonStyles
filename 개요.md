현재 위치 계산 -> useCurrentLocaton 커스텀훅
ㄴ한글 주소 변환 -> kakao API
ㄴ날씨 -> openWeatherMap API

날씨 정보 받아온 후 날씨 번호에 WeatherDescription 적용하여 한글 날씨로 변경 -> 맑음 , 태풍 등
(WeatherDescription 파일은 가져도댕냥 프로젝트에서 사용한 파일 적용)
